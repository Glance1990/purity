Installation

Installation is simple. Just download the zip file from Codecanyon and upload it to your website via Wordpress Plugins page.

Go to Plugins->Add New and click Upload Plugin button
Select zip file from your computer and click Install Now button
Activate the plugin

Galleries

After the plugin activation, you will see a new menu item on the left side of your wordpress admin panel (DV-Galleries).

DV-Galleries: All your galleries will be displayed on this page.

Add new gallery: You can use this menu item to create new galleries

Categories: You can use categories to organize your galleries. These categories are also used on filterable grid galleries.

Shortcodes

DV Galleries

You can use this shortcode to add blog style galleries to any post or page.

max: Maximum number of galleries. Pagination is available on blog style galleries. For example if you have 5 gallery and if you select "4", pagination will be actived and your last gallery will be displayed on the second page.

categoryid: If you want to display galleries from a specific category, you can add your category id like on the following example (You can find category ids on DV-Galleries->Categories page);

[dvgalleries max='99' categoryid='5' vertical='no']

To select multiple categories, you should add comas between category ids like on the following example;

[dvgalleries max='99' categoryid='5,8,9' vertical='no']

vertical: There are two types of blog style galleries. Horizontal and Vertical. If you want to select vertical layout, change "no" with "yes" like on the following example;

[dvgalleries max='99' categoryid='' vertical='yes']

dvgalleries shortcode
Categories Ids

DV Carousel

You can use this shortcode to create gallery carousels.

max: Maximum number of galleries.

categoryid: If you want to display galleries from a specific category, you can add your category id/ids. (You can find category ids on DV-Galleries->Categories page);

columns: Number of visible columns. You can use 1, 2, 3 or 4.

autoplay: true or false

duration: duration of autoplay (seconds). You can use only numbers (1,2,3,4 etc.).

dvcarousel shortcode

DV Grid

You can use this shortcode to display your galleries as a grid.

max: Maximum number of galleries.

categoryid: If you want to display galleries from a specific category, you can add your category id/ids. (You can find category ids on DV-Galleries->Categories page);

itemwidth: Minimum Item Width. You can play with this setting to create 1,2,3,4 etc. columns which fits your theme's design.

dvgrid shortcode

DV Grid Filter

You can use this shortcode to display your galleries as a filterable grid.

Filter menu will be populated with your gallery categories automatically.

max: Maximum number of galleries.

itemwidth: Minimum Item Width. You can play with this setting to create 1,2,3,4 etc. column layouts which fits your theme's design.

dvgridfilter shortcode

DV Square

You can use this shortcode to display your galleries as a square grid.

It may be useful especially on small fields like sidebars.

max: Maximum number of galleries.

categoryid: If you want to display galleries from a specific category, you can add your category id/ids. (You can find category ids on DV-Galleries->Categories page);

dvsquare shortcode

Gallery Settings Plugins->DV Gallery Settings

General

Custom CSS: You can use this field to add your own css codes.

Loading Image: This image is displayed before loading of blog and lightbox images. Recommended size is 64x64 px.

Remove Loading Image: If you dont want to use loading image, check this box.

Zoom Icon: This image is displayed on blog style galleries when mouse is moved on to a photo gallery. Recommended size is 32x32 px.

Video Icon: This image is displayed on blog style galleries when mouse is moved on to a video gallery. Recommended size is 32x32 px.

Carousel Right Arrow: Right arrow icon on Carousels. Recommended size is 24x24 px.

Carousel Left Arrow: Left arrow icon on Carousels. Recommended size is 24x24 px.
